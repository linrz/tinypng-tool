# Tinypng Tool

![NPM downloads](https://img.shields.io/npm/dt/tinypng-tools.svg)
## Install
```shell
npm install tinypng-tools -g
```
## Example
![](./example.gif)

## Usage
```shell
tinypng <path> [options]

tinypng --key=your key

tinypng ./images/test.png

tinypng ./images -r

tinypng ./images/test.png --dist=./resources
```
